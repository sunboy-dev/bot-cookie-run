ฉันมีไอเดียที่จะสร้างโปรแกรมที่เป็น UI ที่ใช้ในคอมพิวเตอร์เป็น script ที่รันแล้วมีหน้า UI ให้ใช้และต้องสามารถใช้ได้ทั้ง windows,macOS

ฟังก์ชันที่สามารถทำได้คือ
1. Detection หน้าจอ android emulator 
2. สามารถควบคุมหน้าจอ android emulator แยกอิสระรายหน้าจอได้ตามคำสั่งที่ตั้งไว้
3. มี log รายงานผล

ฉันอยากสร้างโปรแกรมที่ช่วยเล่นเกม Cookie run  ด้วย android emulator ที่สามารถตรวจจับภาพหน้าจอแล้วเล่นให้แบบอัตโนมัติหน้าจอหลักของเกมฉันได้แนบรูปให้แล้ว

Flow การทำงาน
1. เปิดโปรแกรม android emulator เช่น MuMu
2. กดปุ่ม Detection เพื่อตรวจหาหน้าจอโปรแกรม android emulator ที่เปิดอยู่
3. สามารถลบหน้าจอออกจาก list ในโปรแกรมที่เจอได้
4. กดปุ่ม Start bot เพื่อเริ่มใช้งานบอทเล่น้กมให้อัตโนมัติ
5. หากต้องการหยุดสามารถเลือกหยุดได้แยกรายหน้าจอ และมีปุ่ม Stop bot เพื่อหยุดทำงานทุกหน้าจอ

ขอผลลัพธ์เป็น Implementation spec ที่จะเอาสั่งให้ claude cli สร้างโปรแกรมให้ หากมีคำถามไหนที่อยากได้คอนเฟิร์มถามได้เลย
